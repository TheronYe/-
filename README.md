# -这是一个四川大学教育评估辅助填充的脚本
本人在预科阶段嫌每学期一次的教学评估太麻烦了，所以做了这个脚本，希望save my life and time. Hope it would alse be a time-saver for you!

注意事项:

1:登录学生端还是得自己操作, 需要在教育评估那一页面进入想要进行评估的课程并出现评分和选项页面时脚本才会出现;
        
2:本项目完全开源, 希望后面有大佬完善;           #   codex辅助帮大忙, 预科这一年还是没什么时间搞
        
3:脚本不会记录您的隐私信息, 但不确定是否会有坏人偷偷往脚本里下狠料, 我还没搞懂怎么预防这种情况.
