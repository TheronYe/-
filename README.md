# 功能:

进入具体评估页面后自动填充评价项;

单选题尽量选择 A / 优秀 / 非常满意;

分数字段随机填写 95-98;

多选题随机选择多个选项;

自动填写约 20 字课程评价;

可等待 101 秒后自动点击“保存”按钮.

# 使用方法:

1: 下载py文件到指定的文件夹, 后直接在windows powershell里输入

盘: \保存的文件夹\python.exe -u "I:\四川大学教育评估辅助脚本.py"                    (运行但不会自动保存）

盘: \保存的文件夹\python.exe -u "I:\四川大学教育评估辅助脚本.py" --save             (运行并在101秒后自动保存)

打开四川大学教务系统学生端, 登录成功后进入教学评估列表页, 自己选择要评价的课程或老师, 打开具体评估页面后，脚本会自动填充一次。

然后就可以了.

2: 下载四川大学评教助手.js
