# -这是一个四川大学教育评估辅助填充的脚本
本人在预科阶段嫌每学期一次的教学评估太麻烦了，所以做了这个脚本，希望save my life and time. Hope it would alse be a time-saver for you!

# 注意事项:

1:因为本学期(2025~2026年春, 也就是2026年月9号下午3:35)的我已经用这个脚本填完了, 所以最新的改版要到2026~2027学年春才能验证是否成功.要是敢试一试的bro可以试一试, 如果成功了在issue里冒个泡;

2:登录学生端还是得自己操作, 需要在教育评估那一页面进入想要进行评估的课程并出现评分和选项页面时脚本才会出现;
        
3:本项目完全开源, 希望后面有大佬完善;           #   codex辅助帮大忙, 预科这一年还是没什么时间搞
        
4:脚本不会记录您的隐私信息, 但不确定是否会有坏人偷偷往脚本里下狠料, 我还没搞懂怎么预防这种情况;

5:篡改猴的脚本我也会在完善后上传, 那样会更方便一点. 名字是"四川大学评教助手".

# 使用方法:
下载py文件到指定的文件夹, 后直接在windows powershell里输入
I:\python.exe -u "I:\四川大学教育评估辅助脚本.py"                    (运行但不会自动保存）
I:\python.exe -u "I:\四川大学教育评估辅助脚本.py" --save             (运行并在101秒后自动保存)
然后就可以了.
